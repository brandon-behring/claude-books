# Scaffold gaps log

Running log of gaps in [`@brandon_m_behring/book-scaffold-astro`](https://github.com/brandon-behring/book-scaffold-astro) discovered while porting books in this repo. Batched as upstream issues / PRs at phase boundaries (Phase 0, 1, 2, ...).

## Format

Each row: `phase | chapter | LaTeX construct or scenario | what's missing in scaffold | suggested improvement | filed?`

Empty log is fine — the file exists from Phase 0 to frame the dogfooding practice from day one.
